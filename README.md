# 개요

### C# 프로그래밍 학습
* **1줄 소개**: 사칙연산 로직과 실시간 수식 업데이트 기능을 제공하는 윈도우 표준형 계산기 프로그램입니다.
* **사용한 플랫폼**: C#, .NET Windows Forms, Visual Studio, GitHub
* **사용한 컨트롤**: Button, TextBox (txtInputWindow, txtOutputWindow), Label
* **사용한 기술과 구현한 기능**:
    * 공통 UI 업데이트 메서드(`UpdateOutputWindow`) 설계를 통한 코드 효율화
    * `Substring()`과 `Length`를 활용한 정밀한 한 글자 삭제(Del) 로직
    * `double.Parse()`를 이용한 고정밀 실수 연산 및 데이터 형변환
    * `Contains()` 및 `IndexOf()`를 활용한 실시간 수식 문자열 분석 및 동기화
    * 플래그 변수(`isOperatorClicked`)를 이용한 입력 상태 제어 및 UX 최적화

# 실행 화면 (과제1)

![과제1 실행 화면](이미지_경로_넣기)

**과제 내용:**
* 계산기의 기본 외형을 디자인하고 숫자 버튼(0-9) 클릭 시 입력창에 숫자가 누적되는 기능을 구현합니다.
* 처음 입력 시 `0`이 중복되지 않도록 처리하고, 연속된 숫자 입력 로직을 완성합니다.

**구현 내용과 기능 설명:**
모든 숫자 버튼을 하나의 이벤트 핸들러(`btnNumber_Click`)에 연결하여 코드 중복을 최소화했습니다. 사용자가 숫자를 누를 때마다 `txtInputWindow`에 텍스트가 추가되며, 연산자를 누르기 전까지 첫 번째 피연산자를 구성하는 기초 단계를 완성했습니다.

**사용한 기술과 구현한 기능:**
* `sender` 객체 캐스팅을 통한 버튼 텍스트 추출
* 문자열 결합 연산자를 이용한 숫자 누적 처리

# 실행 화면 (과제2)

![과제2 실행 화면](이미지_경로_넣기)

**과제 내용:**
* 사칙연산 기호(+, -, x, ÷)와 결과 버튼(=)에 대한 연산 로직을 구현합니다.
* 연산자 클릭 시 현재 입력된 숫자를 저장하고, 두 번째 숫자를 입력받을 준비를 하는 플래그 로직을 추가합니다.

**구현 내용과 기능 설명:**
`switch-case`문을 활용하여 사칙연산의 분기 처리를 구현했습니다. 정수형 연산의 한계를 극복하기 위해 모든 연산 데이터를 `double`형으로 처리하여 소수점 아래 결과값까지 정확하게 산출되도록 설계했습니다. 결과 버튼 클릭 시 이전 피연산자와 현재 값을 결합하여 최종 결과값을 도출합니다.

**사용한 기술과 구현한 기능:**
* `double.Parse()`를 활용한 데이터 형변환 및 산술 연산
* `switch-case` 조건문을 이용한 연산자별 분기 처리

# 실행 화면 (과제3)

![과제3 실행 화면](이미지_경로_넣기)

**과제 내용:**
* 편집 기능인 `del`(한 글자 지우기)과 `CE`(현재 입력 초기화) 기능을 구현합니다.
* 메인창의 변화가 상단 보조창(수식창)에도 실시간으로 반영되는 동기화 로직을 추가합니다.

**구현 내용과 기능 설명:**
사용자가 `del` 키를 누를 때마다 `Substring`을 이용해 마지막 한 글자만 제거하는 정밀 편집 기능을 구현했습니다. 특히 `UpdateOutputWindow` 함수를 별도로 설계하여, 계산이 끝난 결과값(`5 x 2 = 10`) 상태에서도 `del`을 누르면 결과값의 마지막 숫자만 지워지며 보조창 수식이 실시간으로 동기화되도록 윈도우 계산기의 실제 메커니즘을 그대로 재현했습니다.

**사용한 기술과 구현한 기능:**
* `Substring` 메서드를 활용한 문자열 인덱스 제어
* `Contains("=")` 분석을 통한 결과값 모드 실시간 수정 로직

![과제4 실행 화면](이미지_경로_넣기)

**과제 내용:**
* 0으로 나누기 방어 로직과 중복 소수점 입력 방지 등 예외 처리를 강화합니다.

**구현 내용과 기능 설명:**

**사용한 기술과 구현한 기능:**
* `MessageBox.Show`를 활용한 런타임 예외 안내
* 소수점 중복 방지 및 부호 전환 수식 구현

# 배운 내용

* **상태 지향 프로그래밍 이해**: 연산자 클릭 여부나 결과값 도출 여부에 따라 버튼의 기능이 달라져야 함을 인지하고, 플래그 변수를 활용해 프로그램의 상태를 정교하게 제어하는 법을 익혔습니다.
* **문자열 조작 기술의 심화**: `Substring`, `IndexOf`, `Length` 등 다양한 문자열 메서드를 결합하여 사용자의 수정 의도를 실시간으로 UI에 반영하는 동기화 로직의 핵심 원리를 마스터했습니다.
* **예외 처리와 안정성 확보**: 사용자의 잘못된 입력(0으로 나누기, 소수점 중복 등)이 시스템 에러로 이어지지 않도록 사전에 차단하는 방어적 프로그래밍의 중요성을 학습했습니다.
* **공통 모듈 설계를 통한 유지보수**: 여러 버튼에서 반복되는 UI 업데이트 로직을 `UpdateOutputWindow`라는 별도 메서드로 추출하여 코드의 재사용성을 높이고 관리 효율을 극대화하는 경험을 가졌습니다.
* **UX 중심의 디테일 구현**: 단순히 지우는 것에 그치지 않고, 수식창에 남은 이전 기록까지 사용자의 편집에 맞게 실시간으로 재구성하는 디테일이 사용자 경험에 미치는 영향을 깊이 이해했습니다.
